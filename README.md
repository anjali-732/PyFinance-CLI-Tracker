# PyFinance-CLI-Tracker
A robust Command Line Interface (CLI) tool for financial management. It categorizes expenses, calculates budget remaining, and provides a percentage-based spending breakdown using core Python logic
💰 Smart Expense Tracker (CLI)
A lightweight, zero-dependency Python application for personal financial management. This tool allows users to track spending, categorize expenses, and visualize their budget health directly through a command-line interface.

🌟 Key Features
Dynamic Budgeting: Set a monthly budget and track your remaining balance in real-time.

Expense Categorization: Tag expenses (Food, Bills, Travel, etc.) to see exactly where your money goes.

Financial Analytics: Automatically calculates total spending and percentage-based breakdowns.

ASCII Data Visualization: Generates a text-based bar chart to visualize spending intensity by category.

Error Handling: Robust input validation to prevent crashes from invalid data entry.

🛠️ How It Works
The system is built using core Python structures:

Lists & Dictionaries: Used to store and organize expense objects.

Logic Engine: Calculates the percentage of the budget used and maps it to an ASCII-based progress bar.

Clean UI: Uses string formatting to create a readable, tabular summary of your finances.
